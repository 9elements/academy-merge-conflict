In this tutorial you'll learn how to resolve a merge conflict.

We assume you know how git branches are working? In case you don't you should definetly checkout this tutorial.

This is a table of useful git commands:

git commit Commiting something
git branch Work on branch
git checkout Checkout branch/tag/commit
git checkout -b Checkout a new branch
git reset Reset
git revert Revert a commit
git merge Merge with branch
git rebase Rebase with branch
git fetch Fetch from remote
git pull Fetch from remote and merge or rebase
git push Push to remote

In case you have any questions feel free to ask.
